# External Dependencies

This directory is intentionally not tracked in Git because these third-party dependencies contain large simulation assets.

Use the upstream repositories when these environments are needed:

- LIBERO: https://github.com/Lifelong-Robot-Learning/LIBERO
- SimplerEnv: https://github.com/simpler-env/SimplerEnv
- RoboCasa: https://github.com/robocasa/robocasa

Clone or install them locally as needed. The local contents of this directory are ignored by `.gitignore`.
